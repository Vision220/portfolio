# VisionForge Portfolio

A premium dark-themed portfolio website for **VisionForge** (Coimbatore, Tamil Nadu, India) displaying creative graphic design, awards & branding, video editing, photo editing, and event media projects.

## Features

- **Dynamic Projects Showcase**: Filter projects by clicking folder cards.
- **Lightbox Preview**: Full-screen modal for viewing high-resolution designs.
- **Lead Capture Form**: Submits lead details (Name, Contact, Service Needed, and Project Details) directly to a secured **Supabase Database** using Row Level Security (RLS).

## Technologies Used

- **Frontend**: HTML5, CSS3 Vanilla, Vanilla JavaScript.
- **Backend**: Supabase (JS Client via CDN).
- **Hosting**: Vercel.

## Setup & Deployment

1. **Local Setup**:
   Simply open index.html in any web browser.

2. **Deploy to Vercel**:
   - Push to a GitHub repository and link it to Vercel, or
